# Aapnbazaar Website Requirements Document

## 1. Application Overview

### 1.1 Application Name
AapnBazaar

### 1.2 Application Description
AapnBazaar is a next-generation B2B e-commerce platform built to simplify how factories, wholesalers, and shops do business. We connect verified manufacturers directly with buyers, removing unnecessary middlemen and enabling zero-commission trading.\n
### 1.3 Core Value Propositions
- Zero Commission – Businesses keep what they earn
- Direct Factory-to-Shop Network – Faster deals, better pricing
- End-to-End Logistics Support – Safe, reliable goods movement
- Verified Sellers & Buyers – Reduced fraud, increased confidence
- Built for Indian Businesses – Practical, scalable, and local-first\n
### 1.4 Brand Colors
- Primary: #138897\n- Secondary: #29adbd
- Accent: #2796a8

## 2. Website Features

### 2.1 3D View Design
The website should feature a modern 3D visual presentation to showcase the platform's innovative approach and create an engaging user experience.

### 2.2 Vendor Connection Form
A dedicated form for vendors who want to connect through the website, allowing them to register and join the platform.

**Form Fields:**
- Business Name
- Contact Person Name
- Mobile Number
- Email Address
- Business Type (Factory/Wholesaler/Shop)
- Product Categories
- Location
- Additional Information\n
### 2.3 WhatsApp Integration
Direct WhatsApp messaging functionality - when users click the WhatsApp button, it should open a WhatsApp conversation automatically for instant communication.

### 2.4 Contact Section
A comprehensive contact section including:
- Contact form\n- Business address
- Email address
- Phone number
- WhatsApp contact button
- Social media links

### 2.5 App Download Section
Prominent display of the Android app download link:\n- Google Play Store link: https://play.google.com/store/apps/details?id=bajarse.gola.com\n- App features highlight
- QR code for easy download

## 3. Technical Requirements

### 3.1 Database\nUse DynamoDB for database connection to store vendor registration form data and other application data.

### 3.2 Key Sections
- Hero section with 3D elements
- Platform features showcase
- How it works section
- Vendor registration form
- App download section
- Contact section\n- Footer with important links
